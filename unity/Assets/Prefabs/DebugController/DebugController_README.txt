Debug FPX Agent Controller

Scene Setup:
-Enable the DebugFPSAgentController script on the FPS Controller in the scene
-Disable the DiscreteRemoteFPSAgentController script
-make sure the “DebugCanvas” prefab is in the scene and enabled

Controls:

Mouse to look, WASD to move

Point central reticle at objects to see if they are interactable

Left Click - Open targeted Receptacle
Right Click - Close targeted Receptacle

alphanumeric 1, 2, 3... 9, 0 - Press to try and pick up object from corresponding list of currently visible objects on top left of screen. 
                               This also places it in the Inventory. Only one object allowed in inventory at a time.

Space - Place object currently in inventory in targeted Receptacle
